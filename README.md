## Info

Rosta is a surveying app made with React and ASP.Net core.<br/> 

It was initially developed for an olympiad but is now mostly used to show what its developers can do.

## Tools

Front-end
- React
  - react-router
    - connected-react-router
  - react-spring
  - react-i18next
- Redux
  - redux-thunk
- Typescript
- Spectre.css

Back-end
- .Net Core 3.0
- ASP.NET Core
- MySQL
- C#
